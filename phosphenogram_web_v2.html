Phosphenogram

Browser-based phosphene drawing and reconstruction tool. Draw phosphenes, customise the display, upload a still image or animated gif background, and save local PNG/CSV output without installing Python.

Designed and developed by Remy Cohan.

Recommended workflow: set the parameters first, then click Apply settings + start fullscreen drawing. Controls: draw with mouse, trackpad, stylus, or touch. Press S to save PNG+CSV, C to clear, Z to undo, and Esc to leave fullscreen.

GitHub Pages is static hosting. So users can download saved files locally, but the site cannot collect those files back to your server unless you later add a backend or form service.
