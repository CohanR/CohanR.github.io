Instructions

This viewer is pre-loaded with the MNI 152 standard brain. Hover over the brain to see anatomical labels and coordinates.

To load your own image:

  1. Click on File (top left).
  2. Click on Close ALL to clear the current view.
  3. Drag and drop your .nii.gz file (or DICOM folder) onto the viewer area.
  4. Alternatively: Click File → Add Image to browse.
